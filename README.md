## 使用步骤
### 另有tg bot版本在main分支可实现机器人对话：https://github.com/coderDulu/node-chatgpt/tree/main

### 1. git clone https://github.com/coderDulu/node-chatgpt.git

### 2. 创建config.json
~~~json
{
  "apiKey": "your apikey",
  "proxy": "your http proxy"
}
~~~
### 3. npm i 

### 4. npm start

### 5. run of: `http://ip:3200`
